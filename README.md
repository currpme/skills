# skills

This repository contains a collection of skill packs, each stored in its own directory with a `SKILL.md` entry file and any supporting assets or scripts.

## Included skills

- `beyondcent-ppt-skill`
- `doc-coauthoring`
- `docx`
- `frontend-design`
- `pdf`
- `pptx`
- `skill-creator`
- `xlsx`

## Structure

- `SKILL.md` files define the skill behavior and usage notes.
- `assets/`, `scripts/`, `agents/`, and `references/` folders provide supporting resources when needed.
- Some skills include additional docs such as `LICENSE.txt`, `.md`, or schema files.

## Usage

Choose the skill directory you need and read its `SKILL.md` first. If the skill includes scripts or assets, follow the instructions in that file for the supported workflow.

## Updating

When adding a new skill:

1. Create a new directory for the skill.
2. Add a `SKILL.md` file.
3. Include any required assets, scripts, or references.
4. Update this README if you want the new skill listed here.
